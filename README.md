Library Application | تطبيق المكتبة
A simple React app for displaying and searching through a book library with real-time filtering.
 Features
•	Fetch books from REST API
•	Real-time search functionality
•	Loading states
•	Reload button
•	Clean, responsive UI
 Technologies
•	React (Hooks: useState, useEffect)
•	Axios
•	JSONPlaceholder API

What I Learned
•	Converting Class Components to Functional Components
•	Using React Hooks instead of lifecycle methods
•	API integration with axios
•	State management and filtering
